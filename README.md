# 用车成本计算器 (Vehicle Operating Cost Calculator)

一款简洁实用的网页工具，帮助用户计算和对比油车与电车的用车成本。支持部署在 GitHub Pages 上。

## 功能特性

### 核心计算功能

- **能源费用计算**：支持油车（油耗）和电车（电耗）的能源费用计算，可自定义日/月/年行驶里程
- **保险费用计算**：根据车辆价格、座位数、车龄、险种选择等参数估算年度保险费用，支持用户自定义实际报价
- **车贷计算**：支持等额本息贷款计算，可自定义贷款金额、期限、利率
- **保养费用**：年度保养费用估算，油车和电车默认值不同
- **洗车费用**：月均洗车次数和单次费用计算
- **折旧费用**：基于车辆价格和年折旧率计算，油车默认10%/年，电车默认15%/年
- **违章与过路费**：月均违章和过路费用
- **停车费用**：月均停车费用

### 多年成本对比

- 支持1-10年的用车成本预测
- 考虑保险费用随无出险年限递减（最高50%折扣）
- 考虑折旧费用随车辆价值下降而递减
- 可视化累计成本趋势图

### 数据管理

- **独立数据存储**：油车和电车参数独立保存，切换时互不影响
- **一键同步**：可将当前车型的通用参数同步到另一种车型
- **车辆价格联动**：在任一模块修改车辆价格，自动同步到其他模块

### 用户体验

- **中英文切换**：支持中文和英文界面
- **模块开关**：每个费用模块可独立启用/禁用
- **响应式设计**：适配桌面端和移动端
- **暗色主题导航**：护眼的深色导航栏

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式方案**：Tailwind CSS
- **状态管理**：Zustand
- **路由方案**：React Router (HashRouter，兼容 GitHub Pages)
- **图表库**：Recharts
- **图标库**：Lucide React

## 部署

### GitHub Pages 部署

1. 构建项目：
   ```bash
   npm run build
   ```

2. 将 `dist` 目录内容部署到 GitHub Pages

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
